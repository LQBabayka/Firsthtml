Node JS - Быстрый Курс за 1 час

После установки последней версии nodeJS уже можно использовать фишки nodeJS в обычной консоли:
npm -v это чекнуть установленную версию (node package manager)
node Это переход к работе с JS из терминала (есть нюансы с функциями)
npm init Это инициализация проекта. Используется для создания файла package.json

В терминале VS code чтобы запустить конкретный файл перед его наименованием необходимо набрать Node (node main.js)

По ходу просмотра видео устанавливаю пакеты для нода:
npm i chalk - форматирование сообщений, выводимых в терминал (цвета логов консоли)
npm i nodemon -D - Онлайн перезагрузка сервера после внесенных изменений для удобства. Флаг -D указывает, что это дополнение необходимо только для разработчика
